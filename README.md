# LED team Warehouse

Sistem za upravljanje magacinom za LED team (Serbia).

## Podešavanje Firebase

1. Otvori `src/firebase.js`
2. Idi na Firebase Console → Project Settings → Your apps
3. Ako nema app-a, klikni "</>" (Web app) i registruj
4. Kopiraj `firebaseConfig` i zalepi u `src/firebase.js`

## Pokretanje lokalno

```
npm install
npm start
```

## Deploy na Vercel

1. Push na GitHub
2. Import na vercel.com
3. Deploy!
